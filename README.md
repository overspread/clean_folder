# Clean Folder Tool

这是一个简单高效的目录清理工具，旨在帮助用户快速删除文件夹中的空文件和空文件夹，同时绝对保护包含内容的有用数据。

## 🚀 功能特点

- **安全清理**：仅删除大小为 0 的空文件，绝不触碰有内容的文件。
- **递归删除**：支持递归清理空文件夹。如果一个文件夹内部的子文件夹被删空后，该文件夹也变为空，则会一并删除。
- **跨平台兼容**：提供 `.sh` 脚本及其配套的 Windows `.bat` 启动器。
- **智能路径转换**：支持直接输入 Windows 格式路径（如 `D:\test`），自动转换为 Linux 识别格式。
- **全中文界面**：所有提示和交互均使用中文。

## 🛠️ 使用方法

### 在 Windows 上运行
1. 双击运行 `run_clean.bat`。
2. 根据提示输入要清理的文件夹完整路径（例如：`D:\my_folder`）。
3. 从以下选项中选择清理模式：
   - `1`: 仅递归删除空文件。
   - `2`: 仅递归删除空文件夹。
   - `3`: 同时递归删除空文件和空文件夹。

### 在 Linux/Git Bash 上运行
```bash
chmod +x clean_dir.sh
./clean_dir.sh
```

## ⚠️ 注意事项
- **不可逆操作**：删除操作是永久性的，请在运行前确认路径正确。
- **环境要求**：Windows 用户需安装 Git Bash 或 WSL 等支持 Bash 脚本的环境才能运行 `.bat` 文件。

## 📂 文件说明
- `clean_dir.sh`: 核心清理逻辑脚本。
- `run_clean.bat`: Windows 环境下的快捷启动文件。
